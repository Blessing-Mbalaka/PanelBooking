# UJ Panel Booking Platform

A Django-based booking system for university panel presentations with supervisor-student matching, one-booking-per-person restrictions, and real-time capacity tracking.

## Features

✅ **Multi-day panel scheduling** with date, panel, and slot management  
✅ **Student-Supervisor matching** with conflict detection (same-panel prevention)  
✅ **Supervisor autocomplete** search with live filtering  
✅ **One booking per person** with cancellation + reason tracking  
✅ **Identity validation** - names and emails must be registered  
✅ **CSV export** of all bookings  
✅ **Django admin** for data management  
✅ **Production-ready** with Gunicorn + Whitenoise  

## Tech Stack

- **Backend**: Django 6.0.5
- **Database**: SQLite (local) / PostgreSQL (production)
- **Frontend**: Vanilla JavaScript (no dependencies)
- **Server**: Gunicorn
- **Static Files**: Whitenoise
- **Deployment**: Render

## Local Setup

### 1. Clone & Install

```bash
git clone <your-repo>
cd BookingPlatform
python -m venv .venv
.venv\Scripts\activate        # Windows
source .venv/bin/activate     # macOS/Linux
pip install -r requirements.txt
```

### 2. Create .env

```bash
cp .env.example .env
# Edit .env with your SECRET_KEY (optional for local dev)
```

### 3. Initialize Database

```bash
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```

Visit: `http://localhost:8000/`

### 4. Load Supervisor Data

If you have a CSV of supervisors and students:

1. Go to `/admin/` and upload via **SupervisorStudentLink → Upload Excel**
2. Then run: `python manage.py load_supervisors extract`

Or manually: `python manage.py load_supervisors add`

## API Endpoints

### Booking Management

**GET** `/api/bookings/`  
List all active bookings

**POST** `/api/bookings/`  
Create booking (requires: firstName, surname, email, role, date, panel, slot, supervisorName)

**POST** `/api/bookings/<id>/cancel/`  
Cancel booking (requires: email, reason)

**DELETE** `/api/bookings/`  
Clear all bookings (admin only)

### Schedule & Counts

**GET** `/api/schedule/`  
Get all days, panels, and slots

**GET** `/api/system-counts/`  
Get total students and supervisors from registration

### Supervisors

**GET** `/api/supervisors/search/?q=name`  
Search supervisors by name (≥2 chars, top 20 results)

**GET** `/api/supervisor-links/upload/`  
Upload CSV/Excel with supervisor-student mapping

**GET** `/api/supervisor-links/template/`  
Download CSV template

### Data Export

**GET** `/exportdata/`  
Download all bookings as CSV

### Admin

**GET** `/admin/`  
Django admin panel (requires superuser login)

## Database Models

### ScheduleDay
- `date` (Date, unique)

### Panel
- `day` (FK to ScheduleDay)
- `name` (CharField, unique per day)

### Slot
- `day` (FK to ScheduleDay)
- `role` (student|supervisor)
- `label` (e.g., "10:00 AM")
- `sort_order` (integer)

### Booking
- `first_name`, `surname`, `email`
- `role` (student|supervisor)
- `supervisor_name` (for students only)
- `day`, `panel`, `slot` (FKs)
- `status` (active|cancelled)
- `cancellation_reason`, `cancelled_at`
- `booked_at` (auto_now_add)
- **Constraint**: Unique (day, panel, role, slot) when status=active

### Supervisor
- `name` (CharField, unique)
- `email` (EmailField, optional)

### SupervisorStudentLink
- `supervisor_name`, `supervisor_email`
- `student_name`, `student_email`
- **Constraint**: Unique (supervisor_name, student_name)

2. Upload file with columns: supervisor, supervisor_email, student, student_email

1. Go to `/admin/` → Supervisor
2. Click **Add Supervisor**

## Troubleshooting

**"No matching query exists" when booking**
- Ensure your name and surname are registered in SupervisorStudentLink
- If no mappings uploaded, the identity check is skipped

**"Supervisor not found"**
- Run `python manage.py load_supervisors extract` after uploading mappings
- Or manually add via `/admin/` → Supervisor

**"Slot taken" or permission errors**
- The slot may already be booked (check /api/bookings/)
- Or there's a supervisor-student conflict (same panel restriction)
